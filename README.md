## About

This is a useless little script that adds [rickrollrc](https://github.com/keroserene/rickrollrc) as aliases for `yarn` and `npm` to the users `.bash_profile`, `.bashrc` and `.zshrc`

## Usage

```sh
$ npx rickrollme
```
